# timule-hw-ctl
Timule HW Controller

## OLED

[DIYmall 0.96" 128x64 White I2C][1]

### Connecting to RPi

VCC - 1
GND - 6
SDA - 3
SCL - 5


### Install OLED library

```sh
cd src/ArduiPi-SSD1306
sudo make
```
